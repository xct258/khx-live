使用说明
📌 工具简介
这是一个专为括弧笑直播回放设计的视频合并工具，支持从多个视频中提取精彩片段并合并为一个完整视频，所有代码均使用AI生成，如有bug欢迎私信反馈。
🚀 快速上手
  1. 选择视频：首先需要选择视频文件
  2. 预览：点击"预览"按钮加载视频
  3. 标记片段：播放到目标位置后，使用 Q 设定起点，W 设定终点
  4. 添加片段：点击"确认添加片段"或按 C
  5. 合并：重复上述步骤添加多个片段，最后输入用户名并点击"开始合并"
⌨️ 完整快捷键
空格播放 / 暂停 ← / →后退 / 前进 1 秒 Q设定片段起点 W设定片段终点 C确认添加当前片段 P预览当前选中的片段 [跳转到片段起点（方便微调）
✨ 高级功能
  • 预览合并：在提交前预览所有片段的合并效果并调整顺序
  • 历史输出：查看和管理之前生成的合并文件
  • B站投稿：直接将合并后的视频投稿到哔哩哔哩
  • 多视频合并：可从不同视频中提取片段并合并为一个文件
  • 进度实时显示：合并时显示实时进度和预计剩余时间
⚠️ 注意事项
  • 同一时间只能运行一个合并任务
  • 片段列表和用户名会自动保存在浏览器中
  • 合并成功后当前片段列表会自动清空
