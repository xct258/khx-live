使用说明
📌 工具简介
本工具用于从括弧笑直播回放中选择片段、预览顺序并合并成一个成品视频。当前支持视频时间轴切片、字幕切片、跨视频片段合并、片段导入导出、历史输出管理和 B 站投稿。
🚀 推荐流程
  1. 选择视频：点击“选择视频”，在目录树中进入文件夹并选择需要切片的预览视频。
  2. 选择切片方式：勾选“启用视频预览”使用时间轴切片；有字幕时可勾选“使用字幕切片”。
  3. 添加片段：用时间轴按钮或快捷键设置起点/终点，也可以在字幕列表中点“+”添加字幕片段。
  4. 检查列表：打开“待合并片段”，预览全部、调整顺序、删除或导出备份。
  5. 提交合并：填写用户名称和可选文件名，选择“原版”或“压制版”，点击“开始合并”。
🎬 视频选择与配置
  • 目录树：只展示可用于预览的录播视频；进入文件夹后可以返回上级目录。
  • 启用视频预览：加载主播放器和时间轴，适合精确找点，但会消耗流量。
  • 使用字幕切片：当前视频存在字幕时可用；关闭视频预览也能通过音频预览字幕片段。
  • 源版本：合并时“原版”使用无压制源文件，“压制版”使用带压制效果的投稿版源文件。
📐 时间轴切片
  • 加载预览：选择视频后点击预览区的“预览”，等待播放器可播放后再使用快捷键。
  • 设置片段：点击 Start [Q] 设置起点，点击 End [W] 设置终点；起点和终点都存在时会自动添加片段。
  • 手动输入：也可以在“片段工坊”中直接输入起点和终点，再点“确认添加片段”。
  • 时间轴工具：支持全图/多倍率缩放、跟随播放、缩略图、音频波形、删除所选片段。
  • 片段预览：选中时间轴片段后点 ▶Play，或使用“预览全部”检查最终顺序。
📝 字幕切片
  • 字幕列表：选择视频后若找到字幕文件，勾选“使用字幕切片”即可显示字幕列表。
  • 播放/定位：点击字幕可跳转位置；点 ▶ 可播放该字幕对应的视频或音频片段。
  • 添加/移除：点 + 将字幕时间段加入待合并片段；已加入或重叠时会显示为可移除状态。
  • 编辑字幕：可编辑字幕起止时间和文本，保存后会写回字幕文件并记录变更。

📋 片段列表管理
  • 待合并片段：显示所有视频的片段，支持删除单个片段和拖拽排序。
  • 跨视频合并：可以添加来自不同视频的片段，合并时会按列表顺序处理。
  • 预览全部：按待合并顺序连续播放；未启用视频预览时会尽量使用音频预览。
  • 自动保存：片段列表和用户名会保存在当前浏览器中，刷新后会提示是否恢复。
  • 清空：会删除当前浏览器中的待合并片段，操作前会二次确认。
📦 导入与导出
  • 导出：将当前待合并片段保存为 clips_*.json，适合备份或转移到其他浏览器。
  • 从文件导入：点击“导入”后选择“从导出的文件导入”，再选择之前导出的 JSON 文件。
  • 从元数据导入：选择“从成品视频元数据导入”，粘贴成品视频中的 comment 元数据或 ffprobe 输出。
  • 替换规则：导入会替换当前片段列表；如果当前已有片段，会先弹窗确认。
🔨 合并任务
  • 时长限制：单次最终合并总时长不能超过 90 分钟。
  • 命名规则：输出文件会使用“用户名称-文件名称-时间戳”的格式；文件名称可留空。
  • 任务队列：服务端同一时间只处理一个合并任务，其他任务会排队等待。
  • 进度弹窗：显示当前阶段、全部片段进度、预计剩余时间和取消按钮。
  • 完成后处理：合并完成后可下载、复制链接，并选择清空或保留当前片段列表。
📺 历史输出与投稿
  • 历史输出：合并成功后会记录到本机浏览器，可在“历史输出”中下载、复制链接或删除记录。
  • 文件存在检查：如果服务器上的成品文件已被删除，下载和复制按钮会自动禁用或提示失败。
  • B 站投稿：历史输出中可打开投稿弹窗，填写标题、简介、封面和标签后提交。
  • 上传进度：投稿过程中会显示百分比、速度、剩余时间和日志，也可以停止当前投稿。
⚠️ 注意事项
  • 同一视频内的片段不能互相重叠；相邻片段可以首尾相接。
  • 正在切片、合并或排队时，片段添加、删除、排序等编辑操作会受到限制。
  • 视频预览、缩略图和波形都会产生额外请求，网络较慢时建议减少同时开启的辅助功能。
  • 自动保存、历史输出和“不再提示”只保存在当前浏览器；换设备或清理浏览器数据后不会保留。
  • 成品视频会写入切片元数据，后续可通过“导入 → 从成品视频元数据导入”恢复片段列表。
  • 工具仍在持续迭代，如遇到异常，请保留错误提示、视频名和操作步骤方便排查。
